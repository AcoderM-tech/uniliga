# Uniliga — Frontend

React 18 + Vite frontend for the Uniliga football league platform.

## Setup

```bash
npm install
```

Configure `.env.local`:

```env
VITE_API_URL=https://yourdomain.com
VITE_WS_URL=wss://yourdomain.com
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```

Built with Docker as part of the full stack — see root `docker-compose.yml`.
